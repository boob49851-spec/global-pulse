# Global Pulse - نسخة الهاتف المحسّنة

## 🌍 نظام مراقبة عسكري متقدم

نظام مراقبة شامل لتتبع الأحداث والنزاعات والأخبار العالمية في الوقت الفعلي، محسّن بالكامل للعمل على الهواتف الذكية والأجهزة اللوحية.

---

## ✨ الميزات الرئيسية

### 🗺️ الخريطة المباشرة
- عرض تفاعلي للأحداث العالمية على خريطة ثلاثية الأبعاد
- تدوير الخريطة باللمس بسهولة
- عرض تفاصيل الأحداث عند النقر عليها

### 📰 البث الوارد المباشر
- أخبار مباشرة من مصادر موثوقة
- تصفية حسب الفئات (صراع، نزاع، سياسية، اقتصادية، قتالية)
- البحث عن أخبار محددة بالكلمات المفتاحية
- تصدير البيانات (JSON/CSV)

### 📊 البث المباشر
- بث مباشر من قنوات إعلامية مختلفة
- التبديل بين القنوات بسهولة
- دعم YouTube والقنوات الأخرى

### 📈 التحليلات
- إحصائيات الأحداث
- مؤشرات التهديد الإقليمية
- تحليل الاتجاهات

### 📱 ميزات الهاتف
- **ملاحة سفلية**: 5 أزرار سهلة الوصول
- **قائمة درج**: خيارات إضافية
- **PWA**: تثبيت كتطبيق أصلي
- **العمل بدون إنترنت**: تخزين محلي ذكي
- **الإشعارات**: تنبيهات فورية للأحداث المهمة

---

## 🚀 البدء السريع

### المتطلبات
- Node.js 14+ و npm
- متصفح حديث (Chrome, Firefox, Safari, Edge)

### التثبيت

```bash
# استنساخ المشروع
git clone <repository-url>
cd global-pulse4_mobile

# تثبيت المتطلبات
npm install

# بدء السيرفر
npm start
```

السيرفر سيعمل على `http://localhost:3000`

### الوصول من الهاتف

1. **على نفس الشبكة**:
   ```
   http://<your-computer-ip>:3000
   ```

2. **عبر الإنترنت**:
   - استخدم خدمة ngrok أو Cloudflare Tunnel
   - أو انشر على Vercel/Netlify

## 📱 التثبيت كتطبيق على الهاتف

### Android
1. افتح المشروع في Chrome
2. اضغط على ⋮ (القائمة)
3. اختر "تثبيت التطبيق"
4. اضغط "تثبيت"

### iOS
1. افتح المشروع في Safari
2. اضغط على ↗ (مشاركة)
3. اختر "إضافة إلى الشاشة الرئيسية"
4. اختر الاسم واضغط "إضافة"

## 🎮 الاستخدام

### الملاحة الأساسية
| الزر | الوظيفة |
|------|--------|
| 🗺️ خريطة | عرض الخريطة المباشرة |
| 📰 أخبار | عرض البث الوارد |
| 📅 جدول | الجدول الزمني |
| 📊 تحليلات | التحليلات والإحصائيات |
| ☰ قائمة | خيارات إضافية |

### اختصارات لوحة المفاتيح
| المفتاح | الوظيفة |
|--------|--------|
| 1-4 | التبديل بين الأعمدة |
| Esc | إغلاق القائمة |
| Cmd/Ctrl + K | البحث |

### تفاعلات اللمس
- **السحب على الخريطة**: تدوير الخريطة
- **النقر على حدث**: عرض التفاصيل
- **السحب من اليمين**: فتح القائمة
- **النقر خارج القائمة**: إغلاق القائمة

## 🔧 البنية التقنية

### الملفات الرئيسية

```
├── index.html              # الصفحة الرئيسية
├── styles-v2.css          # الأنماط الأساسية
├── styles-mobile.css      # أنماط الهاتف
├── app-v2.js              # منطق التطبيق
├── mobile-nav.js          # مدير الملاحة
├── sw.js                  # Service Worker
├── manifest.json          # ملف PWA
├── server.js              # السيرفر Express
└── data.js                # البيانات الثابتة
```

### التقنيات المستخدمة
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **WebSockets**: للبث المباشر
- **PWA**: Service Workers, Manifest
- **3D Graphics**: Globe.gl
- **RSS Parser**: لتحليل الأخبار

## 🐛 استكشاف الأخطاء

### المشكلة: الملاحة لا تظهر
```javascript
// تحقق من وحدة التحكم
console.log(window.mobileNav);
```

### المشكلة: الخريطة لا تعمل
```javascript
// تحقق من تحميل Globe.gl
console.log(window.GlobeGL);
```

### المشكلة: الإشعارات لا تعمل
```javascript
// تحقق من Service Worker
navigator.serviceWorker.getRegistrations();
```
